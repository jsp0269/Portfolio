EL Collector

  • 기획 의도 : 학교 건물을 3D로 모델링하여 내부를 돌아다닐 수 있는 시뮬레이션
  • 기        간 : 2015.06.29 ~ 2015.07.31
  • 총 인원수 : 5명
  • 개발 도구 : Unity 3D
  • 사용 언어 : C#, Javascript
  • 담당 업무 : 스크립트 개발
  • 개발 파트
    • 전체 레벨의 Director 스크립트 개발
    • UI 스크립트 개발
    • Object 스크립트 개발
    • Scene 프로세스 구성

EXPLANATION

  • 시뮬레이션 모드와 게임 모드를 선택하여 플레이할 수 있습니다.
    • 시뮬레이션 모드 : 학교를 자유롭게 돌아다닐 수 있는 모드입니다.
    • 게임 모드 : 학교를 돌아다니며 제한 시간 내에 랜덤한 위치에 스폰되는 5개의 EL동아리 로고를 찾는 모드입니다.
  • 게임 모드에서는 난이도를 조정하여 플레이할 수 있습니다.
    난이도는 총 3단계로 Easy, Normal, Hard로 구성됩니다.
  • 비교적 친숙한 1인칭 방식을 선택하여 키보드와 마우스로 플레이가 가능합니다.
    게임모드에서는 마우스의 왼쪽 클릭을 통해 로고를 획득할 수 있습니다.
