SafeBox

  • 기획 의도 : Arduino를 이용한 택배보관함 시스템
  • 기        간 : 2018.05.29 ~ 2018.06.12
  • 총 인원수 : 1명
  • 개발 도구 : Arduino IDE
  • 사용 언어 : C
  • 사용 H/W
    • 아두이노 UNO
    • 화염감지 센서
    • 압력감지 센서
    • 초음파 센서
    • LCD 디스플레이
    • 서보 모터
    • RGB LED
    • 부저

EXPLANATION

  • 키패드를 통한 패스워드 입력을 지원합니다.
  • 비밀번호 일치 시 LCD 디스플레이, LED 점등을 통해 알리며 서보모터가 회전하여 잠금이 해제됩니다
  • 비밀번호 오류 시 LCD 디스플레이, LED 점등을 통해 알립니다.
  • 내부 화재 발생 시 센서를 통해 인식, 부저와 LED 반복 점등을 통해 알립니다.
  • 지면과 10cm 이상 이격 시 부저가 지속적으로 울려 도난을 방지합니다.
  • 문이 닫힐 시 압력 센서가 감지하여 자동으로 잠금을 실행합니다
