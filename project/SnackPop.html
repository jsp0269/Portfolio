PROJECTS

Snack Pop

  • 기획 의도 : 같은 과자 3개를 한줄로 만들어 없애는 퍼즐 게임
  • 기        간 : 2015.12.31 ~ 2016.01.30
  • 총 인원수 : 5명
  • 개발 도구 : Unity 3D
  • 사용 언어 : C#, Javascript
  • 담당 업무 : 기획, 스크립트 개발
  • 개발 파트
    • Game 스크립트 개발
    • Scene 프로세스 구성

EXPLANATION

  • 제한 시간 내에 과자를 연달아 3개 이상 이어지게 하여 없애는 게임입니다.
  • 플레이 횟수 및 최고 점수 5위까지 랭킹에서 볼 수 있습니다.
  • 게임 종료 시 스코어를 보여줍니다.
  • 일정 시간동안 과자를 못찾을 시 힌트를 제공합니다.
  • 한 번에 4개 이상 제거하게 될 경우 폭탄 과자가 됩니다.
  • 폭탄 과자를 제거할 시 그 줄을 모두 제거합니다.
  • Android 플랫폼에서 이용할 수 있습니다.
