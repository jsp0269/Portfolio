PROJECTS

EscapeRoom

  • 기획 의도 : VR 기기를 이용한 가상현실 방탈출 구현
  • 기        간 : 2019.05.28 ~ 2019.06.14
  • 총 인원수 : 2명
  • 개발 도구 : Unity
  • 사용 언어 : C#
  • 담당 업무 : 스크립트 개발
  • 개발 파트
    • 전체 Scene의 Director 스크립트 개발
    • UI 스크립트 개발
    • Object 스크립트 개발
    • 플레이어 스크립트 개발
    • Scene 프로세스 구성

EXPLANATION

  • PC에서 사용 가능한 대다수의 VR기기 호환이 가능하도록(높이 등) 개발하였습니다.
  • 인지부조화 해결을 위하여 텔레포트 기능을 이동 방식으로 사용하였습니다.
  • 오른손의 컨트롤러를 이용하여 도구를 획득, 사용할 수 있습니다.
  • 도구 및 반응하는 오브젝트를 상속 관계로 개발하여 오브젝트 반응을 구현하였습니다.
  • 총 16단계의 풀이과정으로 구성되어 있으며 비밀번호 입력, 선택 등의 과정은 매 게임마다 랜덤으로 지정됩니다.
