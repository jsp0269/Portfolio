더 지니어스 - 결합

  • 기획 의도 : tvN 프로그램 '더 지니어스'에 나온 결합 게임을 안드로이드에서 즐길 수 있는 게임
  • 기        간 : 2019.10.25 ~ 2019.10.29
  • 총 인원수 : 1명
  • 개발 도구 : Unity 3D
  • 사용 언어 : C#

EXPLANATION

  • tvN 프로그램 '더 지니어스'에 방송된 결합 게임을 그대로 재연하였습니다.
  • 생소한 이용자를 위해 게임 방법을 제공합니다.
  • 랭킹에서 플레이 횟수와 20위까지의 랭킹을 제공합니다.
  • 랭킹은 초기화할 수 있습니다. 이때 플레이 횟수도 초기화됩니다.
  • 합 성공 시 +1점, 결 성공 시 +3점입니다.
  • 합, 결 실패 시 -1점입니다.
  • 단 점수가 0점일 경우 감점되지 않습니다.
