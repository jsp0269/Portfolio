스마트 제가습기

  • 기획 의도 : 디스플레이를 통해 온/습도를 확인하고 제습, 가습을 제어할 수 있는 스마트 제가습기
  • 기        간 : 2019.10.01 ~ 2019.10.24
  • 총 인원수 : 5명
  • 개발 도구 : PyCharm, Arduino IDE, Raspbian OS
  • 사용 언어 : Python, C
  • 사용 H/W
    • Raspberry Pi
    • Arduino Nano
    • 릴레이 모듈
    • 온습도 센서
    • TFT 터치스크린 (3.5inch)
    • LED

EXPLANATION

  • 디스플레이를 통해 온습도를 확인할 수 있습니다.
  • 디스플레이 터치를 통해 제습, 가습을 실행할 수 있습니다.
  • 일정 습도 이상/이하가 될 경우 자동으로 제습/가습 기능을 실행합니다.
  • 디스플레이 터치를 통해 LED를 제어할 수 있습니다.
