Farm Car

  • 기획 의도 : 자율주행을 이용하여 넓은 농지에 대한 자동화 관리가 가능한 시스템
  • 기        간 : 2018.11.20 ~ 2018.12.21
  • 총 인원수 : 3명
  • 개발 도구 : Arduino IDE
  • 사용 언어 : C
  • 담당 업무 : 알고리즘 및 스크립트 개발
  • 사용 H/W
    • 아두이노 UNO
    • 블루투스 모듈
    • 적외선 감지 센서
    • 적외선 LED
    • 서보 모터
    • 워터 펌프
    • 릴레이 모듈
    • 추가 배터리

EXPLANATION

  • 간편한 구동을 위해 스마트폰 App을 통해 원격 제어가 가능합니다.
    • 1. 일반주행 : 참새 등을 쫓기 위한 허수아비 대체 주행 기능
    • 2. 물 분사 : 주행을 하며 물 또는 농약을 분사하는 기능
    • 3. 밭 갈기 : 밭 내부를 주행하며 전방 하단에 위치한 서보 모터가 동작하여 밭을 가는 기능
  • 적외선 LED 및 감지 센서를 통해 전방 및 좌우측의 물체를 감지합니다.
  • 하단 서보 모터를 작동시키면 이동하며 밭을 갈게 됩니다.
  • 워터 펌프를 작동시키면 이동하며 물을 뿌립니다.
